# Shortcut-Downloader
Shortcut Downloader using Windows shortcut

Example:

shortcut.py -p payload -l link -s savename -i icon
#
shortcut.py -p download -l https://the.earth.li/~sgtatham/putty/latest/x86/putty.exe -s test.exe -i 1
